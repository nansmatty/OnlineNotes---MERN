This is a simple notes app. Where you can log in and create notes, update notes and also delete notes. You have your own profile where you can change your profile details and even change your profile picture.
In this app, I used React.js as frontend [ Redux for state management ], and as backend, I used Node.js and for database MongoDB.
As for authentication I used the JSON Web Token.
This app is also deployed on Heroku.
You can check this here https://onlinenotes-mern.herokuapp.com/
